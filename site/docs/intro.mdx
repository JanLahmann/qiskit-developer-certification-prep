# Start here

You want the **IBM Qiskit v2.x Developer certification**. This platform exists to get you from "registered" to "passed" — with a guided path, practice questions that are *machine-verified* against real Qiskit, and a mock exam that faithfully mirrors the real thing.

:::info Unofficial
This is an independent community project — not affiliated with, endorsed by, or sponsored by IBM. All practice content is generated from the public exam objectives and open documentation, never from actual exam content.
:::

## The exam at a glance

| | |
|---|---|
| **Exam** | C1000-179 — *Fundamentals of Quantum Computing Using Qiskit v2.X Developer* |
| **Certification** | IBM Certified Quantum Computation using Qiskit v2.X Developer — Associate |
| **Questions** | 68 (multiple choice, some multi-select) |
| **Time** | 90 minutes |
| **Pass** | 47 of 68 (≈69%) |
| **Covers** | Qiskit SDK v2.x + Qiskit Runtime primitives (SamplerV2 / EstimatorV2) |
| **Register** | [Official exam page at IBM Training ↗](https://www.ibm.com/training/certification/ibm-certified-quantum-computation-using-qiskit-v2x-developer-associate-C9008400) |

## The eight exam sections

Your study is organized around the official syllabus — eight sections, weighted like the exam itself:

| # | Section | Weight | ≈ Questions |
|---|---|---|---|
| 1 | Perform quantum operations | 16% | ~11 |
| 2 | Visualize circuits, measurements, and states | 11% | ~7 |
| 3 | Create quantum circuits | 18% | ~12 |
| 4 | Run quantum circuits | 15% | ~10 |
| 5 | Use the sampler primitive | 12% | ~8 |
| 6 | Use the estimator primitive | 12% | ~8 |
| 7 | Retrieve and analyze results | 10% | ~7 |
| 8 | Operate with OpenQASM | 6% | ~4 |

Section 3 + Section 1 alone are a third of the exam — plan accordingly.

## How to use this platform

1. **Learn** — each section page curates the official IBM resources (documentation, IBM Quantum Learning courses, videos) for its objectives. We link to the sources the exam is actually based on.
2. **Drill** — practice questions per section, each with proof-backed explanations: why the right answer is right (executed and verified against a pinned Qiskit 2.x stack) and why each wrong answer is wrong.
3. **Check** — take the mock exam: 68 questions in 90 minutes, sampled to match the official section weights, pass line at 47. Your score report shows exactly which sections need work, and what to do about it.

Your progress is stored only in your browser (localStorage) — no account, no tracking, exportable anytime.
